Agri vision Ai

About the Project :
AgriVision AI is a unified Machine Learning + Deep Learning smart agriculture system built to support farmers, agritech users, and AI-based crop monitoring in one deployable application.

It combines structured-data prediction modules and image-based crop analysis modules to solve real farming problems such as crop selection, irrigation planning, fertilizer guidance, yield estimation, crop health monitoring, disease detection, pest detection, nutrient deficiency detection, NDVI analysis, and crop stress prediction.

The project is designed as an end-to-end agritech decision-support platform where:
ML modules handle prediction and recommendation tasks using farm input data.
DL modules handle crop image analysis and vegetation health monitoring.
Both are integrated into one AgriVision AI combined app for practical deployment.

Project Goal
The main goal of AgriVision AI is to help users make better farming decisions through AI-powered intelligence. It supports:
Choosing the right crop
Predicting weather-related farming conditions.
Planning irrigation correctly.
Managing fertilizer needs.
Detecting crop diseases and pests early.
Monitoring plant health using image-based analysis and NDVI.
NDVI Analysis.
Crop Stress Prediction.
